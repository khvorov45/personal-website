Running RStudio on an Ubuntu server over SSH

5 Nov, 2019 - 2 minutes
The idea RStudio server listens to port 8787 while it’s running on the server. We need to establish an SSH tunnel from a local port (e.g. 1234) to the remote server’s port 8787. Once that’s done (instructions below), open a web browser, type it localhost:yyyy where yyyy is a local port the tunnel’s been set up to forward to (e.g. 1234). RStudio login page will greet you. Use you server’s username and password.
